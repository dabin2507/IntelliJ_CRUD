# IntelliJ_CRUD
스프링 부트 게시판

## 🖥️ 프로젝트 목표
게시판 DATE CURD + REST API + Test Code + JavaScript 활용한 댓글 REST API
<br>

## 🕰️ 개발 기간
* 23.9월 ~ ing

### ⚙️ 개발 환경
- `Java 11`
- `JDK 1.8.0`
- **IDE** : IntelliJ
- **Framework** : Springboot(2.x)
- **Database** : H2

## 📌 주요 기능
#### 사내 게시판, 자유 게시판
- 글 작성, 읽기, 수정, 삭제(CRUD)
- 댓글 작성, 수정, 삭제
- 파일 다운로드
